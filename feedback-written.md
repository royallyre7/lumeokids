# User Feedback — LumeoKids

- **How collected:** Chat messages — 2 users (Thuta + anonymous)
- **When:** 2026-07-04

## Raw feedback

1. "UI is very simple and not attractive. Need to improve the assessment function. Should add more functions or tests to check child's strength and weakness."
2. "Can't edit or delete child profiles — once created, there's no way to change anything. Needs update and delete functionality."
3. "Assessment results are shown but there's no way to share or export them. Would be useful to download as PDF or share with teachers."
4. "No way to track progress over time — if a child takes the assessment again, old results are overwritten instead of showing history."

## Themes (what keeps coming up)

- **UI needs polish** — feels too simple for a kids' app
- **Profile management incomplete** — no edit/delete after creation
- **Assessment could be deeper** — more test types, export, history

## Top 3 things to fix

- [ ] Add edit and delete child profiles
- [ ] Improve UI attractiveness (icons, animations, consistency)
- [ ] Add more assessment test types (logic, creativity, memory)
