# User Interview — LumeoKids

- **Who:** Thuta (friend)
- **When:** 2026-07-04
- **How:** Chat message — ~10 min

## What they do today (without your project)

Uses generic learning apps for kids but finds them one-size-fits-all. No way to track individual strengths or understand how their child learns best.

## What they liked

- Assessment function is really good — the 10 sections are clear and the archetype results make sense
- Easy to register and create a child profile

## What confused them / what's missing

- Can't edit or delete a child profile once it's created — that's frustrating
- UI feels rough and inconsistent in places
- No way to export or share assessment results

## What would make them actually use it

- Ability to update and delete profiles
- Polished, more attractive UI
- More assessment test types to better evaluate strengths and weaknesses

## What I'll change (next steps)

- [ ] Add edit child profile feature
- [ ] Add delete child profile with confirmation dialog
- [ ] UI consistency pass across all pages
